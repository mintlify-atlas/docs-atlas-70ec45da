## What is Killa?

Killa is a **small and lightweight state management library** for vanilla JavaScript and React, inspired by [Zustand](https://github.com/pmndrs/zustand) and [SWR](https://swr.vercel.app). It provides a simple, subscription-based model for managing application state with minimal boilerplate.

```bash
npm install killa
```

## Key features

<CardGroup cols={2}>
  <Card title="Zero boilerplate" icon="wand-magic-sparkles">
    Create a store with a plain object. No reducers, no actions, no providers required.
  </Card>
  <Card title="Selector subscriptions" icon="filter">
    Subscribe to only the slices of state you care about. Components only re-render when their selected state changes.
  </Card>
  <Card title="Immutable state" icon="lock">
    Internal state is immutable — all reads return deep clones, preventing accidental mutations.
  </Card>
  <Card title="Persist middleware" icon="floppy-disk">
    Persist state to `localStorage` or any custom storage with optional encryption and auto-revalidation.
  </Card>
  <Card title="Vanilla & React" icon="code">
    Works in plain JavaScript, React, React Native, and Next.js environments.
  </Card>
  <Card title="Custom middleware" icon="puzzle-piece">
    Extend your store with custom middleware using the `use` option in `createStore`.
  </Card>
</CardGroup>

## Quick example

<Tabs>
  <Tab title="Vanilla JS">
    ```javascript
    import { createStore } from 'killa'

    const store = createStore({ counter: 0 })

    // Subscribe to state changes
    store.subscribe((state, prevState) => {
      console.log('Counter:', state.counter)
    })

    // Update state
    store.setState((state) => ({
      ...state,
      counter: state.counter + 1
    }))

    store.getState() // { counter: 1 }
    ```
  </Tab>
  <Tab title="React">
    ```jsx
    import { createStore } from 'killa'
    import { useStore } from 'killa/react'

    const store = createStore({ counter: 0 })

    function Counter() {
      const [state, setState] = useStore(store, (s) => ({
        counter: s.counter
      }))

      return (
        <button onClick={() => setState((s) => ({ ...s, counter: s.counter + 1 }))}>
          Count: {state.counter}
        </button>
      )
    }
    ```
  </Tab>
  <Tab title="CDN (Browser)">
    ```html
    <script src="https://unpkg.com/killa@1.10.0/dist/umd/killa.min.js"></script>
    <script>
      const { createStore } = window.killa
      const store = createStore({ counter: 0 })
    </script>
    ```
  </Tab>
</Tabs>

## Browser & runtime support

Killa supports React >= 18.0.0, Chrome 58, Firefox 57, IE 11, Edge 18, Safari 11, and Node.js >= 16.
